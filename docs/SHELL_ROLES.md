# RTG Shell Roles

## Purpose

RTG shell roles provide a way to classify the dynamic role of a transition cell.

The symmetry is inspired by s, p, d, and f shell behavior in chemistry, but RTG shell roles are not claims about electron orbitals. They are governance-dynamics analogies.

## Provisional Shell Roles

### s-like cells

Foundational stabilizers.

```text
core validity
identity
receipt integrity
minimum provenance
repeatable low-ambiguity checks
```

### p-like cells

Directional translators.

```text
domain-to-domain relation
policy-to-evidence alignment
authority-to-action translation
observer-to-state orientation
```

### d-like cells

Coordination mediators.

```text
multi-party transitions
variable authority states
quorum behavior
dispute handling
escalation
quarantine
replay mediation
```

### f-like cells

Deep complexity cells.

```text
latent dependencies
long-range coupling
hidden context
emergent behavior
slow stabilization
high observability burden
```

## Site Presentation Direction

Possible visual grammar:

```text
s = compact / stable / core glow
p = directional / lobed / oriented
d = faceted / multi-node / coordination geometry
f = layered / shadowed / deeply coupled
```

## Core Principle

> The transition table should not only show where a cell is; it should show what kind of transition-orbital the cell occupies.
