# RTG Semantic Differentiation Phase 1

## Purpose

This phase begins the transition from broad template-coherent tests to mechanism-specific RTG behavior.

## Prior State

The 250-test baseline established a broad executable grammar:

```text
observation
identity compatibility
authority
receipt / replay
lineage / export
confidence
quarantine
finality
maturity boundary
```

## New State

This bundle adds 12 tests whose classifier logic intentionally differs by mechanism.

## Mechanisms Added

```text
observer_window_semantic_behavior_tests
authority_boundary_semantic_behavior_tests
false_transition_semantic_behavior_tests
unknown_unknown_semantic_behavior_tests

dark_cell_color_semantic_behavior_tests
quarantine_release_semantic_behavior_tests

supersession_lineage_semantic_behavior_tests

```

## Core Transition

```text
250 tests → broad executable grammar
262 tests → broad executable grammar + first semantic mechanism split
```

## Non-Claim

This phase does not complete semantic differentiation.

It establishes the first differentiated layer pattern.
