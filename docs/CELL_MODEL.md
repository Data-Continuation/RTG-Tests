# RTG Cell Model

## Assumption

A transition cell is a local admissibility geometry, not a static table entry.

## Provisional Cell Parameters

```text
cell_id
    stable identifier for the transition cell

label
    human-readable cell name

maturity
    draft/speculative/provisional/tested/proven/deprecated

observation
    how much governed observation exists for the cell

evidence_density
    how much evidence has accumulated

confidence
    confidence posture derived from admissible observations

admissibility_pressure
    pressure applied by policy, authority, risk, evidence demand, and context

risk_resistance
    resistance created by risk, ambiguity, unresolved dependencies, or hazard

stabilization
    degree to which the transition has converged into a coherent state

emission_readiness
    degree to which the cell can emit governed knowledge

darkness
    unobserved or underclassified remainder

function_class
    the cell behavior under observation/admissibility pressure

shell_role
    dynamic role class inspired by s/p/d/f shell symmetry

governance_posture
    accepted/rejected/deferred/quarantined/superseded/unresolved/contested/unknown
```
